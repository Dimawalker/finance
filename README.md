Консольное приложение для учета личных финансов с авторизацией пользователей, бюджетированием и аналитикой расходов.

## Возможности

### Базовый функционал
- **Авторизация** - регистрация и вход пользователей
- **Учет операций** - добавление доходов и расходов
- **Бюджетирование** - установка лимитов по категориям
- **Статистика** - детальные отчеты по категориям
- **Оповещения** - уведомления о превышении бюджета
- **Сохранение** - автоматическое сохранение данных в JSON

### Дополнительные возможности
- **Help система** - встроенная справка по командам
- **Категории** - автоматическое создание категорий
- **Баланс** - расчет текущего финансового состояния
- **Фильтрация** - просмотр данных по категориям

**Архитектура

src/main/java/com/finance/
├── Main.java                      # Точка входа
├── model/                         # Модели данных
│   ├── User.java                  # Пользователь

│   ├── Transaction.java           # Транзакция
│   ├── TransactionType.java       # Тип операции
